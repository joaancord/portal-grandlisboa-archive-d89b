# portal-grandlisboa-archive-d89b

本仓库用于归档和发布多个独立 HTML 页面。这些页面本身不针对任何特定域名或具体网站，而是作为通用页面存档集合存在。

## 项目简介

这是一个以存档为目的的仓库，存储了若干独立的 HTML 文档。这些文档可能包含信息展示、静态内容或其它非交互式的页面设计。仓库本身不提供动态服务，仅作为页面文件的版本管理与公开访问的载体。

## 目录说明

```
.
├── index.html          # 仓库主页或入口页面（如有）
├── pages/              # 存放主要 HTML 页面的目录
├── assets/             # 静态资源文件（图片、样式、脚本等）
└── README.md           # 本文件
```

- `pages/`：包含所有归档的 HTML 页面文件，按需组织子目录。
- `assets/`：存放页面引用的外部资源，如 CSS、JavaScript、图片等。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开浏览。
- 页面内容可能涉及多种主题，但均为历史存档或静态展示用途。
- 不保证页面内容与任何外部网站存在关联或同步更新。
- 归档的页面可能包含过时信息或链接，仅供查阅参考。
